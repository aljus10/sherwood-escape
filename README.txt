Sherwood Escape
================

What it is:
A small original browser game inspired by the fast arcade feel, but redesigned around a Robin Hood vs ninja chase.

Files:
- index.html
- style.css
- game.js

How to run:
1. Extract the zip.
2. Open index.html in a browser.

Controls:
- Up / W: sprint
- Down / S: brake
- Left / A: move left
- Right / D: move right
- Space: fire arrow
- R: restart after game over

Notes:
- No build step needed.
- Everything is plain HTML, CSS, and JavaScript.
- Works best in a modern Chromium browser.
